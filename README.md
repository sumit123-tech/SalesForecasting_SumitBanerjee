# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes Superstore sales data to identify business trends, customer behavior, regional performance, product performance, and sales insights using Python and data visualization techniques.

The project includes data cleaning, exploratory data analysis (EDA), business KPI generation, visualizations, and business recommendations.

---

## 🎯 Objectives

- Analyze sales performance across categories and regions.
- Identify top-selling products and customers.
- Study customer purchasing behavior.
- Analyze shipping performance.
- Generate business insights and recommendations.

---

## 📂 Dataset

- Dataset: Superstore Sales Dataset
- Records: 9,800 Orders
- Format: CSV

## 💡 Key Business Insights

- Technology generated the highest revenue.
- Phones were the best-selling sub-category.
- West region contributed the highest sales.
- Tuesday and Saturday recorded peak sales.
- Standard Class was the most frequently used shipping mode.
- A small number of customers generated a significant portion of total revenue.

---

## 🚀 Business Recommendations

- Increase inventory for Technology products.
- Promote high-demand products like Phones and Chairs.
- Improve marketing in the South region.
- Reward loyal customers.
- Optimize shipping operations.
- Run promotional campaigns during low-sales periods.
